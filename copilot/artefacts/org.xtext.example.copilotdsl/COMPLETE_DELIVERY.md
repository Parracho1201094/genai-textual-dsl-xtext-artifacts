## 🎓 GETTING STARTED GUIDE

### Step 1: Understand What This Is
LTS is a textual DSL for modeling Excel-like ETL (Extract, Transform, Load) processes. It generates executable Java code.

### Step 2: Read Quick Start
→ Open **QUICKSTART.md**

### Step 3: See an Example
→ Open **examples/Salary.lts**

### Step 4: Learn the Syntax
→ Read **doc/SYNTAX_GUIDE.md**

### Step 5: Set Up Eclipse
- Install Eclipse IDE 2021.12+
- Install Xtext 2.26+ from Marketplace
- Ensure Java 11+ is configured

### Step 6: Import Projects
- File → Import → Existing Projects
- Select all 4 projects
- Click Finish

### Step 7: Generate Artifacts
- Right-click org.xtext.example.copilotdsl
- Run As → Generate Xtext Artifacts
- Wait for completion

### Step 8: Test It
- Right-click org.xtext.example.copilotdsl.ui
- Run As → Eclipse Application
- Create test.lts and copy Salary.lts content
- Test syntax highlighting and validation
