// 12. EXECUTION INSTRUCTIONS FOR ECLIPSE
// ============================================================================

To run the LTS DSL in Eclipse IDE:

Step 1: Setup Eclipse
--------
- Install Eclipse IDE 2021.12 or later
- Install Xtext 2.26 via Eclipse Marketplace
- Ensure Java 11+ is configured

Step 2: Import Projects
--------
a) File → Import → Existing Projects into Workspace
b) Select workspace folder
c) Select all 4 projects:
   - org.xtext.example.copilotdsl
   - org.xtext.example.copilotdsl.ui
   - org.xtext.example.copilotdsl.tests
   - org.xtext.example.copilotdsl.ide

Step 3: Generate Xtext Artifacts
--------
a) Right-click org.xtext.example.copilotdsl
b) Run As → Generate Xtext Artifacts
c) This runs GenerateMyDsl.mwe2 and creates:
   - ANTLR parser
   - Ecore metamodel
   - Infrastructure classes
   - Validator stubs
   - Generator stubs

Step 4: Build Project
--------
a) Project → Clean → Clean all projects
b) Wait for automatic build
c) Verify no errors (may have warnings)

Step 5: Launch Test Instance
--------
a) Right-click org.xtext.example.copilotdsl.ui
b) Run As → Eclipse Application
c) This opens a new Eclipse window with LTS support

Step 6: Test with Examples
--------
a) File → New → Project → General → Project
b) Create test project
c) Create new file: test.lts
d) Copy content from examples/Salary.lts
e) Observe:
   - Syntax highlighting works
   - Content assist available (Ctrl+Space)
   - Validation shows results
   - Errors highlighted in red
   - Warnings highlighted in yellow

Step 7: Generate Java Code
--------
a) Right-click test.lts
b) Run As → Generate Java from Xtext Model
c) Check src-gen folder for generated code
d) Inspect generated SalaryProcessProcessor.java

Step 8: Run Generated Code
--------
a) Compile generated code with Apache Commons CSV
b) Prepare input CSV files in data/ directory
c) Run generated processor
d) Check output/ folder for results
e) Review logs/salary_process.log
