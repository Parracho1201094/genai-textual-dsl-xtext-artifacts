# Runtime Java Demo

This folder is immediately runnable and demonstrates the Java code that the Xtext generator should emit.

## With Maven

```bash
mvn test
./run-all.sh
```

## With only a JDK

```bash
./run-all-javac.sh
```

On Windows PowerShell:

```powershell
.\run-all-javac.ps1
```

The generated outputs appear under `out/`.
